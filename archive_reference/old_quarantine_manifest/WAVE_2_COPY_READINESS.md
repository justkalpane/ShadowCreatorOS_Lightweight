# WAVE 2 COPY READINESS

## Safe to copy into lightweight repo
- directors
- agents
- sub_agents
- skills
- registries
- schemas
- validators
- docs
- rules/contracts/prompts (if present)
- workflow JSONs as REFERENCE ONLY
- PRD/architecture/build-status docs
- dossier templates/examples (selected)

## Forbidden to copy as active runtime
- n8n profile/state folders
- SQLite runtime DB
- binaryData runtime artifacts
- node_modules
- .env and credentials
- live workflow IDs as active truth
- live webhook IDs as active truth
- runtime caches/logs as active state
- OpenWebUI runtime DB/config as active truth
- Gemini-first runtime state as default
