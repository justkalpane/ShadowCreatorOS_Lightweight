# MAC-05 Execution Not Allowed Yet

## Scope

MAC-05 is planning/documentation/design only.

## Explicitly Not Allowed

- n8n runtime start/install/import/execution
- Gemini or provider API calls
- OpenWebUI start/use
- Old Windows runtime usage
- Credential creation/exposure
- Media artifact generation claims

## Decision Gate

Execution can start only after:

1. MAC-05 planning review approval
2. Explicit execution-phase authorization
3. Tooling and policy gates checked
