# MAC-06 Execution Not Allowed Yet

MAC-06 in this phase is planning only.

## Not Allowed

- Executing reviewer-agent workflows
- Starting n8n runtime
- Installing n8n
- Importing or executing n8n workflows
- Calling Gemini or provider APIs
- Starting OpenWebUI
- Generating media artifacts (audio/video/image/avatar)
- Activating old Windows runtime paths

## Allowed

- Documentation
- Review-lane contracts
- Prompt templates
- Scorecard standards
- Planning reports

Execution can begin only after explicit user approval in a later phase.
