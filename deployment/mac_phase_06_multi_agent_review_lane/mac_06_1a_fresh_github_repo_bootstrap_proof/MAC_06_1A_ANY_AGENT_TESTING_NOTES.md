# MAC-06.1A Any-Agent Testing Notes

## Operating Truth

Single-agent operation is primary. Multi-agent review is optional enhancement. n8n/provider execution remains a future layer.

## First Representative Proof

The first proof can be Codex because Codex is the current available repo-aware agent. A successful Codex proof validates one representative repo-aware single-agent path.

## Later Compatibility Tests

ChatGPT, Claude, Kimi, DeepSeek, Gemini, Perplexity, Antigravity, local LLM apps, and future agents can be tested later with the same starter prompt.

Users are not required to install all agents.
