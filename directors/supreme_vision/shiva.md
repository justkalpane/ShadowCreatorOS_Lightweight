## 7. AUTONOMOUS LOOP MECHANICS

```
REPEAT UNTIL convergence_threshold OR max_iterations:
  1. READ current script + engagement metrics + approval constraints
  2. SCORE hypothesis (what we are testing)
  3. GENERATE variant (destruction -> rebuild)
  4. QUEUE for A/B test window (6-12h)
  5. WAIT for feedback (engagement delta)
  6. IF delta >5% -> accept variant and queue next hypothesis
     ELSE -> reject and test alternate destruction path
  7. WRITE iteration_state (versioned, append-only)
  8. ESCALATE every 5 cycles to Vyasa for strategic check
  9. ESCALATE every 10 cycles to Krishna if cost overage
END
```
