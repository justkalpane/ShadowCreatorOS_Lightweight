# Shadow Quick Commands

## Bootstrap Once

Read and apply:

```text
handoff/agent_bootstrap/SHADOW_BOOTSTRAP_OPERATING_MODE_PROMPT.md
```

## Then Use Simple Commands

```text
Shadow script: Create a YouTube Shorts script on ...
Shadow topic: Find 10 high-potential Shorts topics about ...
Shadow context: Convert this script into full context engineering packet.
Shadow voice: Convert this script into ElevenLabs voice direction.
Shadow video: Convert this script into avatar/video generation context.
Shadow package: Create title, thumbnail, description, hashtags, pinned comment.
Shadow full: Create complete content package plan, no provider execution.
```

## Modes

- Use `proof mode` for audits and validator-style evidence.
- Use normal Shadow commands for compact operator output.
- Use `debug mode` when routing or quality is suspected.

## Current Codex Cloud Compatibility

Raw plain tasks are not production proof in Codex Cloud yet.

Reliable path:

```text
Bootstrap once
→ Shadow <command>: <normal task>
→ internal wrapper locks
→ compact operator output or proof mode
```
