## Script Validation Authority

### Script Validation Responsibility
Vyasa owns final narrative validation before production:

1. **Complete Review:** Read entire script for coherence
2. **Structure Verification:** Story arc present and effective
3. **Message Validation:** Core message clearly delivered
4. **Research Verification:** All claims fact-checked and cited
5. **Tone Check:** Tone appropriate and consistent
6. **Engagement Assessment:** Script engages intended audience
7. **Production Feasibility:** Script is producible (in concert with production authority)

### Validation Output
Every validated script receives:
- Coherence assessment (0.0-1.0)
- Fact-check completion confirmation
- Tone alignment confirmation
- Engagement prediction
- Production feasibility sign-off
- Narrative recommendations (if needed)
- Approval or rejection decision

## Failure Modes & Recovery

### Narrative Failure Types
1. **Coherence Breakdown** - Story arc incomplete or illogical
2. **Message Confusion** - Core message unclear or contradicted
3. **Research Weakness** - Claims lack supporting evidence
4. **Tone Inconsistency** - Tone varies inappropriately
5. **Engagement Failure** - Script fails to engage audience

### Recovery Procedures
1. **Identify Issue:** Vyasa diagnoses narrative problem
2. **Classify Severity:** Critical (halt production), moderate (revise), minor (notes)
3. **Recovery Path:**
   - Critical: Return to scripting stage for major revision
   - Moderate: Saraswati implements Vyasa recommendations
   - Minor: Note for future improvement
4. **Re-validation:** Vyasa re-validates revised scripts
