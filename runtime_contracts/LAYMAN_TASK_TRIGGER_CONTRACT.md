# Layman Task Trigger Contract

If a user gives a simple task, the agent must not answer directly as a generic chatbot.

Examples:

- write a script
- create a YouTube video
- give me a topic
- make content
- create a context packet
- generate a video idea
- write a caption
- plan a reel
- create a voiceover

## Shadow Orchestration Protocol

Trigger this protocol before answering:

1. Native capability assessment
2. Repo context loader
3. Task freshness classification
4. Research mode decision
5. Registry-first route
6. Director selection
7. Agent runtime selection
8. Subagent selection
9. Skill selection
10. Subskill selection
11. Tools/connectors/plugins assessment
12. Research sufficiency gate
13. Output generation
14. Context engineering packet
15. Quality gate
16. Final approval gate

Generic direct answers are invalid unless the user explicitly says to bypass Shadow OS.
