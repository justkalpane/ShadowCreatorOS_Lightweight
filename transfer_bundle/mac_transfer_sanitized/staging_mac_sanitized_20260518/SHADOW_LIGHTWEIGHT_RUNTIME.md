# SHADOW LIGHTWEIGHT RUNTIME

## Purpose
This repository is the lightweight repo-first intelligence runtime for Shadow Creator OS.

## Strategic Decision
The old Shadow Empire environment is quarantined as future enterprise infrastructure.

## Core Doctrine
- Repo = constitution.
- AI coding agents = intelligence runtime.
- Dossiers = state and truth record.
- n8n = future external execution bus only.
- Providers = optional future media/action layer.

## What this repo does
- topic intake
- research planning
- director selection
- agent/subagent selection
- skill/subskill selection
- script generation
- debate
- critique
- refinement
- final script
- context engineering packet
- provider handoff packet
- quality gate
- lineage record

## What this repo does not do by default
- does not use n8n for intelligence
- does not call Gemini by default
- does not require OpenWebUI
- does not execute paid providers
- does not claim media generation without artifacts
- does not use old workflow IDs as active truth
- does not use old webhook IDs as active truth
- does not use old SQLite as runtime

## Runtime rule
Every user mission creates one dossier folder.

## Provider rule
For ElevenLabs, HeyGen, Higgsfield, Sora, Seedance, YouTube, analytics:
Generate provider handoff packets only unless execution is explicitly requested and runtime credentials are available.

- OpenWebUI is optional.

- Gemini is not default.

- n8n is not the intelligence layer.
