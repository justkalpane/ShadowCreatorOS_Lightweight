# Transfer Package Contents And Exclusions

## SAFE TO TRANSFER
- lightweight repo core
- handoff docs
- deployment docs
- runtime contracts
- directors/agents/subagents/skills/registries/schemas/validators/docs
- sanitized n8n workflow exports
- n8n inventories/checksums/manifests
- audit reports

## NOT SAFE BY DEFAULT
- raw private n8n workflow exports
- old n8n DB/profile/binaryData
- .env/secrets/tokens/cookies
- execution data/logs
- node_modules
- OpenWebUI runtime state
- provider credentials
- old Codex sessions
